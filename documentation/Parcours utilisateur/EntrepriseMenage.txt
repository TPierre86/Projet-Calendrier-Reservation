Arrive sur la page d’accueil du calendrier où il doit entrer ses identifiants
→ Le calendrier s'affiche, mais il ne peut pas interagir avec
→ Le calendrier affiche les événements déjà créés

→ Se connecte à son profil

Bascule sur la page du calendrier :
→ Le calendrier s'affiche, il peut désormais interagir avec
→ Le calendrier affiche les événements 

Clique sur une case vide :
→ Ne peux rien faire 

Clique sur un événement existant (seulement événement "MENAGE"):
→ Peut dire que l'événement est finis
→ Peut commenter l’événement
→ Peut télécharger les fichiers PDF joints à l’événement

Il peut, depuis le calendrier, l’exporter au format Excel ou Google Calendar

Depuis le calendrier, il peut naviguer :
→ Vers les mois suivants
→ Vers les mois précédents
→ Basculer l’affichage du calendrier en semaines
→ Basculer l’affichage du calendrier en jours
→ Revenir à l’affichage initial du calendrier par mois
