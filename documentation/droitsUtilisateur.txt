Gestionnaire :

Son compte est créé par défaut avec des droits administrateur complets.
A accès à l’ensemble du calendrier (toutes les salles et toutes les associations).
Peut voir les commentaires laissés par les associations.
Peut créer des réservations récurrentes pour chaque association (toutes les deux semaines, jour fixe).
Dispose d’un panneau d’administration lui permettant de :
    Gérer les associations : ajouter, modifier, supprimer ;
    Lister les membres de chaque association ;
    Gérer les utilisateurs : ajouter, modifier, supprimer ;
    Gérer les salles : ajouter, modifier, supprimer.
Peut créer des événements "ménage" à destination de l’entreprise de nettoyage.
Peut consulter les commentaires laissés par le personnel de ménage.
Peut ajouter des commentaires :
    Sur les réservations (récurrentes ou ponctuelles) ;
    Sur les créneaux de ménage.
Peut exporter le calendrier global au format Excel ou Google Calendar.

Membre d'association :

doit créer un compte utilisateur en cliquant sur "S’inscrire".
A accès au calendrier de son association.
Peut télécharger les fichiers PDF ajoutés aux événements de son association.
Peut commenter les événements de son association.
Peut exporter le calendrier de son association au format Excel ou Google Calendar.

Président d'association : 

Dispose de tous les droits d’un membre.

Son rôle est attribué par le gestionnaire (il est initialement un membre standard).
Peut créer jusqu’à 3 réservations ponctuelles par mois pour son association.
Peut modifier ou supprimer les réservations de son association.
Peut consulter la liste des réservations en cours de son association.
Peut téléverser des fichiers PDF en plus de commenter les événements.
Peut exporter le calendrier de son association au format Excel ou Google Calendar.

Entreprise de ménage :

Dispose d’un compte déjà créé, avec les droits adaptés à sa fonction.
Peut valider son passage sur les événements de type "ménage" créés par le gestionnaire.
Peut ajouter un commentaire sur les événements de ménage.
Peut voir et commenter les événements organisés par les associations.
